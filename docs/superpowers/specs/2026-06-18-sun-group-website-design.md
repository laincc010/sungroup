## 三、配色系统

### 3.1 CSS 变量定义

```css
:root {
  /* 彩虹强调色 */
  --primary:        #FF6B35;  /* 活力橙 - CTA按钮 */
  --secondary:      #FFD700;  /* 金芒黄 - 数据高亮 */
  --accent-green:   #00D2A0;  /* 翡翠绿 - 通关/海运 */
  --accent-blue:    #4A90D9;  /* 深海蓝 - 空运 */
  --accent-purple:  #7B4FBF;  /* 星云紫 - 仓储 */
  --accent-rose:    #E91E7B;  /* 玫瑰红 - 陆运 */

  /* 深色基底 */
  --bg-primary:     #0A0E17;  /* 主背景 */
  --bg-secondary:   #111827;  /* 卡片背景 */
  --bg-tertiary:    #1A1F2E;  /* 次级区域 */
  --bg-glass:       rgba(255,255,255,0.04);

  /* 文字 */
  --text-primary:   #FFFFFF;
  --text-secondary: #B0B8C8;
  --text-muted:     #6B7280;

  /* 彩虹渐变 */
  --gradient-rainbow: linear-gradient(90deg,
    #FF6B35, #FFD700, #00D2A0, #4A90D9, #7B4FBF, #E91E7B);
}
```

### 3.2 配色原则
- 深色底为主，彩虹色仅用作**点缀和强调**
- 每个业务板块用不同彩虹色作为标签色
- 彩虹渐变用作 section 分隔线和特殊装饰
- 整体配色克制，拒绝花哨

## 六、核心组件规格

### 6.1 导航栏
- `position: sticky; top: 0; z-index: 1000`
- `backdrop-filter: blur(20px); background: rgba(10,14,23,0.85)`
- 滚动后底部出现 1px 彩虹渐变线
- 当前可见 section 对应菜单项高亮 #FF6B35
- 移动端：汉堡图标 → 右侧滑入菜单，带半透明遮罩

### 6.2 Hero Banner
- Swiper 全屏，3 张占位图（物流/港口/货运主题）
- 每张叠加暗角渐变 + 彩虹方向光 overlay
- 标题带 fadeInUp 动画，副标题打字机效果
- 两个 CTA：「立即咨询」(#FF6B35 实心)、「了解更多」(白边框)
- 底部向下滚动箭头（bounce 动画）

### 6.3 玻璃拟态卡片
- 背景 `rgba(255,255,255,0.04)` + `backdrop-filter: blur(10px)`
- 边框 `1px solid rgba(255,255,255,0.08)`
- Hover: translateY(-8px) + 彩虹光晕 box-shadow + scale(1.02)
- 过渡 `0.4s cubic-bezier(0.25, 0.46, 0.45, 0.94)`

### 6.4 数据滚动动画
- 4 项：成立年数、员工数、合作客户、物流覆盖国
- IntersectionObserver 触发
- 数字从 0 递增至目标值（requestAnimationFrame，持续 2s）
- 数字颜色 #FFD700，下方彩虹进度条

### 6.5 荣誉资质灯箱
- 点击缩略图 → 全屏遮罩 + 大图居中
- 键盘左右箭头切换、ESC 关闭
- 点击遮罩空白区关闭
- 缩略图网格 4 列（PC）→ 2 列（手机）

### 6.6 联系表单
- 字段：姓名、电话、需求内容
- HTML5 表单校验（required、type=tel、pattern）
- 提交按钮 #FF6B35，hover 变亮
- 提交后显示成功 toast（右上角滑入，3 秒后消失）
- 右侧：企业信息 + 高德地图 iframe

### 6.7 悬浮快捷栏
- `position: fixed; right: 20px; top: 50%`
- 4 个圆形图标按钮：
  1. 📞 一键拨号
  2. 💬 微信（hover 弹出二维码）
  3. ↑ 返回顶部（滚动超过 300px 后显示）
  4. 💼 在线咨询
- 每个按钮 hover 变为对应彩虹色

### 6.8 页脚
- 顶部彩虹渐变线 2px
- 四列布局：集团简介 / 快捷导航 / 业务板块 / 联系方式
- 底部版权行：© 2026 孙氏集团 + 备案号

## 八、SEO & 性能

- 每个 `<section>` 设置 `id` 属性，利于锚点抓取
- 所有 `<img>` 设置 `alt` 属性
- `<meta>` 标签：title、description、keywords
- CSS 在 `<head>` 加载，JS 用 `defer` 属性
- 图片使用 WebP 格式（降级 PNG）
- 代码注释中文，方便后期修改
